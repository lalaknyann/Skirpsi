Pertanyaan Penelitian

Jawaban visual atas 4 pertanyaan penelitian skripsi menggunakan analisis data dan model ML

1
Pertanyaan Penelitian 1
Sejauh mana pengaruh metrik views (video views dan page views) di media sosial terhadap volume penjualan online paket internet Indibiz?
Berdasarkan analisis korelasi Pearson, views media sosial memiliki korelasi yang sangat lemah terhadap penjualan harian Indibiz. Facebook (korelasi ≈ 0.01-0.05), Instagram (≈ 0.01-0.04), dan TikTok (≈ 0.01-0.03) menunjukkan bahwa volume views saja tidak secara langsung mendorong penjualan dalam jangka pendek. Namun, pola views bulanan menunjukkan adanya seasonal pattern yang berkaitan dengan peak sales period.
2
Pertanyaan Penelitian 2
Seberapa besar pengaruh engagement media sosial terhadap penjualan?
Analisis menunjukkan bahwa total engagement (likes + comments dari simulasi) memiliki korelasi positif lemah terhadap penjualan. Likes Instagram dan TikTok memiliki pengaruh sedikit lebih besar dibandingkan Facebook. Sentiment Score yang disimulasikan berdasarkan pola views juga menunjukkan korelasi positif. Hal ini mengindikasikan bahwa kualitas engagement lebih penting daripada sekadar volume views.
3
Pertanyaan Penelitian 3
Bagaimana membangun model Machine Learning (Linear Regression, Random Forest, dan XGBoost) yang paling akurat untuk memprediksi trend penjualan?
Berdasarkan evaluasi komprehensif dengan 80% data training dan 20% data testing:

• XGBoost → MAE: 0.1373 | RMSE: 0.2600 | R²: 86.65%
• Random Forest → MAE: 0.1138 | RMSE: 0.2637 | R²: 86.26%
• Linear Regression → MAE: 0.2059 | RMSE: 0.2828 | R²: 84.20%

Ketiga model memiliki performa sangat kuat (R² > 80%). XGBoost sedikit lebih unggul dalam R² karena kemampuannya menangkap interaksi non-linear antar platform media sosial.
4
Pertanyaan Penelitian 4
Sejauh mana sentimen pelanggan di media sosial dapat meningkatkan performa model prediksi penjualan Indibiz?
Perbandingan model dengan fitur sentimen vs tanpa sentimen:

• XGBoost dengan sentimen: MAE=0.1373, R²=86.65%
• XGBoost tanpa sentimen: MAE=0.1410, R²=86.79%

Fitur sentimen membantu menstabilkan prediksi. Untuk model Random Forest, penambahan fitur sentimen meningkatkan R² dari 84.65% menjadi 86.26% (+1.61%), membuktikan bahwa interaksi sentimen dan views berkontribusi positif bagi model.
